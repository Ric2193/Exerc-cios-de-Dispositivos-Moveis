# Exercícios de Aquecimento em TypeScript/JS

Este repositório contém a implementação de funções utilitárias para manipulação de arrays, conforme proposto no exercício.

## Funções Implementadas

1.  **unique**: Remove duplicatas de um array.
2.  **groupBy**: Agrupa objetos de um array por uma propriedade específica.
3.  **sumBy**: Soma os valores de uma propriedade numérica em um array de objetos.

## Como Executar

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.

1.  Clone o repositório.
2.  Execute o comando:
    ```bash
    node index.js
    ```
